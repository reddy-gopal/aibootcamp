# Workshop Image Guide

## Current Setup

The pass card uses an image from `/public/workshop-illustration.png`. 

## Image Requirements

**Recommended Specifications:**
- **Size:** 1200x800px (or similar landscape ratio)
- **Format:** PNG with transparency (or JPG)
- **File size:** Under 500KB for fast loading
- **Theme:** AI/Technology workshop focused

## Image Suggestions

### Option 1: Abstract AI Theme
- Neural network patterns
- Circuit board aesthetics
- Digital particles/connections
- Gradient backgrounds with tech elements

### Option 2: Learning Environment
- Modern workspace
- Laptops/computers
- Collaborative learning scene
- Clean, professional look

### Option 3: Futuristic Tech
- Holographic elements
- AI/ML visualizations
- Digital interfaces
- Modern, vibrant colors

## How to Add Your Image

1. **Create or find your image** (1200x800px recommended)
2. **Save it as:** `frontend/public/workshop-illustration.png`
3. **Replace the placeholder file** in that location
4. **The image will automatically appear** in the pass card
